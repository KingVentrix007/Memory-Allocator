\doxysubsection{Detailed Description}
\mbox{\hyperlink{structNode}{Node}} structure for representing a memory region. ~\newline
 

This structure represents a memory region and contains the following fields\+: ~\newline

\begin{DoxyItemize}
\item addr\+: Starting address of the memory region. ~\newline

\item size\+: Size of the memory region. ~\newline

\item next\+: Pointer to the next node in the linked list. ~\newline

\item allocated\+: Boolean flag indicating whether this region is currently allocated. ~\newline

\item first\+\_\+block\+: Boolean flag indicating if this is the first block in the memory region. ~\newline

\item num\+\_\+block\+\_\+used\+: Number of blocks used (will be 0 if first\+\_\+block is false). ~\newline

\item total\+\_\+requested\+\_\+memory\+: Total requested memory for this block of list, not rounded to the nearest 1024. ~\newline
 
\end{DoxyItemize}
