This repository contains a simple memory allocator designed for hobby operating systems. It provides basic functionality for managing memory regions using a linked list data structure.\hypertarget{md_readme_autotoc_md6}{}\doxysection{Features}\label{md_readme_autotoc_md6}

\begin{DoxyItemize}
\item {\bfseries{Memory Allocation}}\+: Allocate contiguous blocks of memory.
\item {\bfseries{Memory Deallocation}}\+: Free previously allocated memory.
\item {\bfseries{Extend Allocation Space}}\+: Dynamically extend the allocation space by connecting to a new memory region.
\item $\ast$$\ast$\+\_\+msize functionality$\ast$$\ast$\+: Provides the ability to get the size of a memory region
\item {\bfseries{Platform independent}}\+: It can run on a hosted environment, or it can run as a part of an OS,
\item {\bfseries{Architecture Independent}}\+: It uses NO architecture specific code, is fully portable
\item {\bfseries{And many others}}
\end{DoxyItemize}\hypertarget{md_readme_autotoc_md7}{}\doxysection{Getting Started}\label{md_readme_autotoc_md7}
