\item {\bfseries{Memory Allocation}}\+: Allocate contiguous blocks of memory.
\item {\bfseries{Memory Deallocation}}\+: Free previously allocated memory.
\item {\bfseries{Extend Allocation Space}}\+: Dynamically extend the allocation space by connecting to a new memory region.
